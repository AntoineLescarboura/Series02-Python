import numpy as np
import matplotlib.pyplot as plt

def sinusoid (amplitude : float, frequency: float, duration: float, num_points: int=1000, phase: float = 0.0):
#     Create a sinusoidal signal

#     Parameters
#     ---------
# float = real number with decimal 

#     amplitud :(maximal amplitude of the sinus)
#     frequency : (frequency in Hz)
#     duration : total time of the signal in seconds
#     num_points : number of the signal points (default is 1000)
#     phase : inital phase in radians 

#     Returns 
#     -------
#     np.ndarray = sinusoid signal evaluated on a linear vector 

    t=np.linspace (0, duration, num_points) # time vector 
    x= amplitude * np.sin(2*np.pi*frequency*t+phase)
    return x,t

A multiple of a number (m) is any integer that can be expressed as the product of (m) and another integer (k).

Mathematically, a number (n) is a multiple of (m) if:

[ n = m \cdot k \quad \text{where } k \text{ is an integer.} ]

Prime or not

A prime number is an integer greater than 1 that has exactly two positive divisors: 1 and itself.

Mathematically, a number (p) is prime if:

[ \text{Divisors of } p = {1, p} ]

In[10]:
def is_prime(n) : 
    if n < 2 :
        return False
    for i in range(2, n) :
        if n % i == 0 :
            return False
    return True
numbers = range(0, 200)  # 0  200   
primes_200 = [n for n in numbers if is_prime(n)]
print(primes_200)  # Affiche les nombres premiers entre 0 et 20
